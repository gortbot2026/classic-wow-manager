Link Your Discord ID

To access guild features, we need to verify your Discord identity.
We'll send a verification code to your Discord DMs.

How to find your Discord User ID
  1. Open Discord (desktop or mobile app)
  2. Go to Settings → Advanced
  3. Enable Developer Mode
  4. Go back and click your username/avatar
  5. Click Copy User ID
  6. Paste it in the field above
